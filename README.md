# Student Record Manager

A simple Student Record Management system built using Node.js, Express, MongoDB, and EJS.  
This project demonstrates CRUD operations along with search and pagination.

---

## Features
- Add new student records
- View all students
- Search students by name, email, or city
- Pagination for large datasets
- MVC architecture
- Clean UI using Bootstrap

---

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap
- Nodemon
